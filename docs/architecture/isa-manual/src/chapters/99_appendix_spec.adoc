[[appendix]]
== Appendix: Spec artifacts and regeneration

[[appendix-artifacts]]
=== Artifacts in this repo

The Linx ISA specification is tracked in multiple forms:

* Golden sources (authoritative): `isa/v0.56/`
* Canonical machine-readable catalog (checked in): `isa/v0.56/linxisa-v0.56.json`
* Generated codec tables: `isa/generated/codecs/linxisa{16,32,48,64}.decode`
* Generated C tables: `isa/generated/codecs/linxisa_opcodes.{h,c}`

[[appendix-regenerate]]
=== Regenerating the catalog and codecs

From the repo root:

[source,bash]
----
python3 tools/isa/build_golden.py --profile v0.56 --pretty
python3 tools/isa/gen_qemu_codec.py --profile v0.56 --out-dir isa/generated/codecs
python3 tools/isa/gen_c_codec.py --profile v0.56 --out-dir isa/generated/codecs
python3 tools/isa/validate_spec.py --profile v0.56
----

[[appendix-build-manual]]
=== Building this manual

From `docs/architecture/isa-manual/`:

make pdf
