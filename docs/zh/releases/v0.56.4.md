# 灵犀指令集 v0.56.4

这是存储库当前实时的规范 LinxISA 配置文件。

## 合约

- 规范 golden 来源：`isa/v0.56/`
- 编译目录：`isa/v0.56/linxisa-v0.56.json`
- Sail 状态：`isa/sail/semantics_status.json`
- ISA 手册：`docs/architecture/isa-manual/`

## 亮点

- 将仓库级规范版本标识从 `v0.56.2` 提升为 `v0.56.4`。
- 将 gather/scatter 文档同步到当前参数模型：
  `validCol` / `validRow` / `Col`、硬件推导的 `Row`、可选 `PadValue`，
  以及按 offset Tile 元素位宽解析的 gather/scatter 偏移量。
- 补全文档中的 `MGATHER.MASK` 与 `MSCATTER.MASK` TMA 功能，并接入
  MkDocs 公共导航。
- 按当前 pin 的 SHA 和严格车道阻塞项刷新 LLVM/QEMU 对齐计划。
