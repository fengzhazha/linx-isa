# 增强指令扩展

增强指令扩展中的指令用于补充32位标准指令的编码长度或者对部分需要多条32位指令组合才能实现的功能进行代码优化。

## 块头指令

增强指令扩展中只包含一类块头指令[HL.BSTART](../header/HL.BSTART.md)。
