# LinxISA v0.56.2

This is the live canonical LinxISA profile for the repository.

## Contract

- Canonical golden sources: `isa/v0.56/`
- Compiled catalog: `isa/v0.56/linxisa-v0.56.json`
- Sail status: `isa/sail/semantics_status.json`
- ISA manual: `docs/architecture/isa-manual/`

## Highlights

- Expands logical tile register depth to 16 entries per hand.
- Uses dynamic tile capacity descriptors.
- Replaces retired pre-v0.56 block descriptor spellings with canonical
  `B.IOT`, `B.CATR`, and `B.DATR` forms.
- Keeps pre-v0.56 material as archive/history only, not as active contract.
