# LinxISA v0.56.4

This is the live canonical LinxISA profile for the repository.

## Contract

- Canonical golden sources: `isa/v0.56/`
- Compiled catalog: `isa/v0.56/linxisa-v0.56.json`
- Sail status: `isa/sail/semantics_status.json`
- ISA manual: `docs/architecture/isa-manual/`

## Highlights

- Promotes the canonical repository-wide profile label from `v0.56.2` to
  `v0.56.4`.
- Syncs the gather/scatter documentation with the current parameter model:
  `validCol` / `validRow` / `Col`, hardware-derived `Row`, optional
  `PadValue`, and width-resolved gather/scatter offsets.
- Documents the new `MGATHER.MASK` and `MSCATTER.MASK` TMA functions and wires
  them into the public MkDocs navigation.
- Refreshes the LLVM/QEMU alignment plan around the currently pinned SHAs and
  active strict-lane blockers.
